Meus Projetos

Atutalmente meu melhor projeto foi o desafio Star Wars, onde o objetivo foi desenvolver uma interface que recebe dados de uma API, as telas deveriam exibir cartões contendo informações dos pilotos e das naves. O projeto foi feito em react e para realização deste projeto foram utilizados: Requisições de API's, Componentes funcionais, hooks, Global states, Router, Styled-components e material MUI.
